# ESF Community Workflow

```
┌─────────────────────────────────────────────────────────┐
│                    YOUR PROJECT                          │
│                                                         │
│  1. POSITION                                            │
│     Write your Position Statement.                      │
│     ┌──────────────────────────────────┐                │
│     │ My stance:          [________]   │                │
│     │ What matters most:  [________]   │                │
│     │ Non-negotiable:     [________]   │                │
│     └──────────────────────────────────┘                │
│     ⚠ No AI until this is written.                      │
│              │                                          │
│              ▼                                          │
│  2. EXPLORE                                             │
│     Bring AI in as a thinking partner.                  │
│     AI challenges your position:                        │
│       • "Have you considered...?"                       │
│       • "What if the opposite were true?"               │
│       • "Your assumption about X may not hold if..."    │
│     You decide what to keep.                            │
│              │                                          │
│              ▼                                          │
│  3. BUILD                                               │
│     Work with AI on the deliverable.                    │
│     For each AI contribution:                           │
│       → Log it (AI Use Log)                             │
│       → Accept, revise, or reject                       │
│       → If you reject or revise:                        │
│         write a Record of Resistance                    │
│              │                                          │
│              ▼                                          │
│  4. VALIDATE                                            │
│     ┌──────────────────────────────────┐                │
│     │ Five Questions:                  │                │
│     │  □ Can I defend this?            │                │
│     │  □ Is this mine?                 │                │
│     │  □ Did I verify?                 │                │
│     │  □ Would I teach this?           │                │
│     │  □ Is my disclosure honest?      │                │
│     └──────────────────────────────────┘                │
│     All pass? → Continue.                               │
│     Any fail? → Go back and fix.                        │
│              │                                          │
│              ▼                                          │
│  5. DISCLOSE                                            │
│     Write an honest disclosure statement.               │
│     Attach it to the finished work.                     │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## The Key Principle

The Position Statement is the gate. Without it, you have no anchor for deciding whether AI output serves your intent or replaces it. Everything downstream depends on knowing what you think before AI tells you what to think.
